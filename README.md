# ai-chat-backend
This project is a backend API for an AI-powered chatbot, built with FastAPI and integrated with OpenAIâ€™s GPT model. ## Features
- Send user messages to the AI chatbot and receive responses
- FastAPI endpoints for chat requests
- CORS enabled for frontend integration
- Example AI responses for testing purposes
- Easy to extend with real data or additional endpoints

## Technologies
- Python 3.13
- FastAPI
- OpenAI API (GPT-3.5 or GPT-4)
- Pydantic (data validation)
- Uvicorn (server)
- JavaScript/HTML frontend for testing (optional)
